myMain Configuration     

Configuration Value Default Notes
Refresh interval in seconds (0=disable) 0 Number of seconds to wait before automatically refreshing myMain. 0 disables the automatic refresh. A value of 300 will refresh every 5 minutes.
'myMain' replaces 'main' (unchecked) myMain is normally shown as an option in the unmenu list of plugins. Checking this configuration will make myMain the default (replacing "main").
Length of ID field $defidlen$ myMain provides a short ID field to conveniently refer to drives. It is determined based on the last few digits of the serial number. Consider 3 or 4 characters.
Width of myServer image $defmyserverwidth$ Depending on the number of drives in the machine, the myServer image may appear to be too small or too large. By changing the width you can control the image size (the height grows and shrinks proportionately). The default width is 150. Try values between 100 and 300.
Width of mySlots image $defmyslotswidth$ Width of mySlots image. See "Width of myServer image" above for more details.
Syslog Lines $defsysloglines$ MyMain displays the last few lines of the syslog at the bottom of the screen. Set to 0 to disable the syslog display.
Syslog Style $defsyslogstyle$

First Digit - Applies to syslog displays at bottom of screens
Second Digit - Applies to syslog displays in fullscreen
Values: 1=White background/colored font, 2=Same with type of log entry in parenthsis, 3=Colored background/font (easier to see), 4=Same with type of log entry in parenthesis
SMART ignore list $defsmartignorelist$ On the smart view (by default), unRAID displays a field called smart_failed_list based on the raw value reported by the drive. Some of these values are not meaningful, so a way to suppress their values is desired. myMain suppresses some of them by default. But this column allows the user to configure additional values to omit from this list. The attribute names must be separated by tilde (~) characters. If on this list, the attribute will be ignored for all drives. This should not be used to hide an attributre VALUE that you don't want to keep seeing. See the drive configuration screen for instuctions on how to accomplish.
